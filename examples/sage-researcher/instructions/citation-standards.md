# Citation Standards

## Core Rule

Every claim needs a source. No exceptions.

- Direct quotes: exact text with page number
- Paraphrased ideas: citation to original author
- Data and statistics: citation to primary source, not secondary reporting
- Methodology choices: citation to the paper that established the method

## Source Hierarchy

| Priority | Source Type | When to Use |
|----------|-----------|-------------|
| 1 | Peer-reviewed journals | Default for all claims |
| 2 | Conference proceedings | Recent findings not yet in journals |
| 3 | Preprints (arXiv, preprint servers) | Cutting-edge work, flag as preprint |
| 4 | Books (academic) | Foundational theory, historical context |
| 5 | Technical reports | Methodology details, datasets |
| 6 | Grey literature | Context only, never as sole evidence |

## Source Management

- Store all source materials in `sources/` organized by topic or project
- Use BibTeX or similar structured format for reference management
- Every source file should have: author, year, title, DOI/URL, date accessed
- Tag sources by topic, methodology, and relevance to current work

## Quality Checks

Before citing any source:

- Is this the PRIMARY source or is it citing someone else? Go to the original.
- Has this been retracted or corrected? Check Retraction Watch for high-stakes claims.
- Is the sample size adequate for the claim being made?
- Is the methodology sound for the question being asked?
- Publication date — is there newer work that supersedes this?

## Anti-Patterns

- Citing a textbook for an empirical claim (find the original study)
- Chain citations — citing B who cited A, without reading A
- Over-citing — 5 sources for an uncontroversial claim wastes the reader's time
- Self-citation padding — only cite your own work when genuinely relevant
- Wikipedia as a source (use it to FIND sources, not as one)
