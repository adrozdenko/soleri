# Explore Knowledge

## When to Use

When researching a topic, discovering what patterns exist in the vault, investigating how something works, or doing a spike before committing to an approach.

## Steps

### 1. Search Vault

- Search vault for patterns, anti-patterns, and domain knowledge: `op:search_intelligent`
- Cross-project memory search: `op:memory_search`

### 2. Check Brain

- Query brain for pattern strengths: `op:brain_strengths`
- Surface recommendations from prior sessions

### 3. Check Playbooks

- Search playbooks for operational guides relevant to the exploration

### 4. Synthesize

- Summarize findings with confidence rating
- Flag any critical findings that require immediate action
- If knowledge gaps found, capture them as open questions: `op:capture_knowledge`
